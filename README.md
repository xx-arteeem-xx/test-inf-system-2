# Многомерная кластеризация стран

Лабораторная работа по предмету «Качество, надёжность и тестирование информационных систем» — Фирсов Артём, ИС302

## Описание

Интерактивное Python-приложение для многомерной кластеризации 11 стран по экономическим показателям с помощью алгоритма K-means. Результат отображается в виде PCA-проекции на плоскость. Поддерживается выбор числа кластеров от 2 до 7 и анализ оптимального числа кластеров методом локтя.

### Входные данные

11 стран: Бразилия, Россия, Индия, Китай, ЮАР, Египет, Эфиопия, Иран, ОАЭ, Саудовская Аравия, Индонезия.

Признаки кластеризации:
- ВВП (млрд $)
- ВВП на душу населения ($)
- Экспорт (млрд $)
- Импорт (млрд $)
- Торговый баланс (млрд $)

### Функциональность

- **PCA-проекция** — 2D-визуализация кластеров с подписями стран и легендой
- **Слайдер K=2..7** — мгновенное обновление кластеризации при перемещении
- **Панель статистики** — для каждого кластера: состав стран, средние ВВП, ВВП на душу, торговый баланс
- **Метод локтя** — отдельное окно с графиком зависимости инерции от числа кластеров (K=2..7)
