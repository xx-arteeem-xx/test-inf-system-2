## [1.0.0] - 2026-06-01

### Added

- **`main.py`** — основной файл приложения. Реализует два класса:
  - `ClusteringApp(tk.Tk)` — главное окно приложения «Многомерная кластеризация»
  - `ToplevelElbow(tk.Toplevel)` — вспомогательное окно «Метод локтя»

- **Данные** — встроенный словарь с экономическими показателями 11 стран (ВВП, ВВП на душу, экспорт, импорт, торговый баланс). Данные стандартизируются через `StandardScaler` перед кластеризацией.

- **K-means кластеризация** — алгоритм `sklearn.cluster.KMeans` с параметрами `random_state=42`, `n_init=10`. Диапазон кластеров: K=2..7.

- **PCA-проекция** — снижение размерности с 5 до 2 компонент через `sklearn.decomposition.PCA`. Результат отображается в виде scatter-графика, встроенного в главное окно через `FigureCanvasTkAgg`.

- **Интерактивный слайдер** — `tk.Scale(from_=2, to=7)`. При перемещении слайдера граф и панель статистики обновляются автоматически без нажатия дополнительных кнопок.

- **Панель статистики** — текстовый виджет `tk.Text` (read-only) справа от графика. Для каждого кластера отображаются: список стран, средний ВВП (млрд $), средний ВВП на душу ($), средний торговый баланс (млрд $). Реализована корректная склонение слова «страна» по-русски.

- **Метод локтя** — кнопка «Метод локтя» открывает отдельное окно `ToplevelElbow` с линейным графиком зависимости инерции KMeans от числа кластеров (K=2..7). Значения инерции подписаны над каждой точкой. При повторном нажатии кнопки окно поднимается на передний план без дублирования.

- **Цветовая схема** — 7 различимых цветов (палитра matplotlib tab10) для раскраски кластеров на scatter-графике.

- **`README.md`** — обновлён: добавлено описание проекта, входных данных, функциональности, инструкции по установке и запуску.

- **`CHANGELOG.md`** — создан (этот файл) в формате Keep a Changelog.
